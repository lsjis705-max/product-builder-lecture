# Diet Management Website Blueprint\n\n## Overview\n\nA web application to help users track their daily food intake and manage their diet, featuring personalized AI-driven feedback and automatic calorie estimation. The application is a single-page application (SPA) built with HTML, CSS, and JavaScript, leveraging modern web standards.\n\n## Implemented Features\n\n*   **Initial Structure:** Basic HTML structure with a header, main content area, and footer.\n*   **Styling:** Modern CSS with a clean and visually appealing responsive design.\n*   **User Profile & Goal Setting:** Users can input and save their height, weight, exercise frequency, and fitness goals.\n*   **Food Entry:** A form to add food items with their name and quantity.\n*   **Upgraded Calorie Estimation Engine:** Accurately estimates calories for common foods and units (e.g., \"김밥 2줄\").\n*   **AI-Powered Diet Feedback:** Provides personalized diet evaluation based on user\'s goals and food log.\n*   **Daily Log:** A list that displays the added food items.\n*   **Calorie Tracking:** Dynamically calculates and displays the total calorie intake.\n*   **Delete Functionality:** Users can remove individual food items from the log.\n*   **Data Persistence:** Uses `localStorage` to save user\'s profile and food log.\n*   **Localization:** The entire UI has been localized to Korean.\n\n## Current Plan: Implement Meal Type
